1. Why is the “Unable to load” appear in the notification bar widget sometimes?

Due to the iOS notification bar has a memory limit, if there is too much content in the notification bar (including today widgets and notifications), there will be problems that cannot be loaded. The solution is to clear the extra notifications and the infrequent widgets in the notification bar. If it still cannot be loaded, restart the phone.

2. Why can’t I scroll in the widget?

Due to system limitations, all pages in the widget can’t be swiped, because if you swiped to scroll will conflict with the system’s gestures, so the iOS disables all swipe gestures in the widget. So we can only rely on the up and down buttons to scroll up and down.

You can not scroll the keyboard association text too.
